Function: default-sizes ()

Return default sizes of a queue with unspecified MAX-MESSAGES and MESSAGE-SIZE in a form of cons (MAX-MESSAGES . MESSAGE-SIZE). This is done by creating a queue with a random name and by extracting its attributes. By using a 255 length name, we protect ourselves from name collision.
